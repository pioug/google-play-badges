# Google Play Badges

This repository contains Google Play badge localizations exported as SVG (January 2025).

- [SVG files](https://github.com/pioug/google-play-badges/tree/main/svg)

## Notes

- Source: https://play.google.com/intl/en_us/badges/
- Filenames in `svg/` now use BCP-47 locale tags (for example: `en.svg`, `pt-BR.svg`, `zh-TW.svg`)
- Outer badge borders were removed from all SVG assets
- Compressed SVG files `svgo -f svg`
